Repositorio principal:
- fsk3-colegio-bohiggins
- URL: https://github.com/crishuinca/fsk3-colegio-bohiggins
- Descripcion: repositorio documental de la entrega EV2. Contiene README general, links, arquetipo Maven y PDFs de patrones y branching.

Repositorios de componentes:

0. eureka-server
- URL: https://github.com/crishuinca/fsk3-eureka-server
- Descripcion: servidor Eureka para registro de ms-academico, ms-asistencia y bff-libroclases.

1. ms-academico
- URL: https://github.com/crishuinca/fsk3-ms-academico
- Descripcion: microservicio Spring Boot encargado de estudiantes, cursos, asignaturas, evaluaciones y notas. Persistencia MySQL (libroclases_academico).

2. ms-asistencia
- URL: https://github.com/crishuinca/fsk-ms-asistencia
- Descripcion: microservicio Spring Boot encargado de asistencias y anotaciones. Persistencia MySQL (libroclases_asistencia).

3. bff-libroclases
- URL: https://github.com/crishuinca/fsk3-bff
- Descripcion: BFF con autenticacion JWT, usuarios, orquestacion via Eureka, Circuit Breaker y docker-compose MySQL compartido (docs/MYSQL.md).

4. frontend-libroclases
- URL: https://github.com/crishuinca/fsk3-frontend-libroclases
- Descripcion: frontend React con login JWT, permisos por rol, paginas de consulta/registro, crear usuario (inspector) y empaquetado como componente NPM.
